A simple single-page web application built with HTML, CSS, and JavaScript that allows users to upload an image, highlight specific areas, and generate 10 glitch effect variations. The app also automatically compiles the generated images into a GIF, giving users the option to download both the individual images and the final animated GIF.

✨ Features
Upload any image from your device.

Highlight specific areas you want to glitch.

Automatically generate 10 unique glitch-effect images.

Combine all generated images into a single animated GIF.

Download all outputs (images + GIF) with a single click.

Runs fully in the browser — no server or external dependencies required.

🛠️ Technologies Used
HTML – Structure and layout.

CSS – Styling and user interface.

JavaScript – Image processing, glitch effects, and GIF generation.

Optionally uses:

html2canvas or <canvas> API for capturing selections.

gif.js, gifshot, or custom canvas logic for GIF creation.

🚀 How to Use
Open the app in your browser.

Upload an image via the file input.

Click and drag on the image to highlight the areas you'd like to apply the glitch effect to.

Click “Generate Glitch Images”.

The app will process and display 10 glitched variations.

A GIF will be automatically created from the images.

Click “Download” on the picture or gif you need
